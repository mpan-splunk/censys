The app_id and secret asset configuration values can be found on the Censys website under the My Account -> API section.

When using the query ip, query domain and query certificate actions, the full Censys search syntax may be used within the query. For examples and syntax help, see the certificate search documentation and similar pages.

Note:

  1. 'Lookup Domain' and 'Query Domain' actions will not work in this version of the app as Censys has not added support for these actions in API V2. For more details refer to pdf attached to this Censys page.
  2. 'Query IP' and 'Query Certificate' action makes 1 API call for every 100 results. For example, to fetch the 5000 results, 50 API calls will be made. If a query is fired for large number of records and the limit specified in the limit parameter is too large, user's account may exceed the query quota provided by Censys.
  3. As per the limit provided by Censys, 'Query Certificate' action can fetch only 25,000 data at max.
